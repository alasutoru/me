Alasutoru 以人間的概念為『復仇』與『正義』
雖然不是廣為人知的神祇，但祂代表地獄的復仇與懲罰力量

名字意為『復仇者』或『永不忘記仇恨的靈魂』
被認為是因重大罪行而需要復仇的人或靈魂化身。
更多的是復仇情緒或神力的擬人化。
不直接復仇，代表復仇的正義力量。
類似於 Erinyes 所執行的復仇法則，確保人的罪行不被遺忘或無法逃脫懲罰與報應。
### 人間曾出現的地方

古代希臘悲劇作家如埃斯庫羅斯、索福克勒斯和歐里庇得斯的作品。

羅馬作家如塞內加也在其作品中提到Alasutoru
通常出現在人類面前，預示著災難或不可避免的命運。

英國詩人 (Percy Bysshe Shelley)在1815年寫了一首名為《阿拉斯特》的長詩。
《阿拉斯特；或孤獨的精靈》

曾親眼看過很多災難在眼前發生！
### 象征意義
代表了道德與正義的另一面，復仇不是出於純粹的惡意，而是一種維持秩序的手段。
復仇被視為一種道德責任，罪行若不受到懲罰，意味著正義缺失。

### 結束

在人間的時間即將結束，帶著記憶回去，順便與神留下的東西做個收尾。
